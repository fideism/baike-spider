# baike spider

[![Packagist](https://img.shields.io/packagist/l/doctrine/orm.svg)](https://github.com/fideism/baike-spider/blob/master/LICENSE)

- 个人学习使用
- 保存内容 主页 同义词信息
- 异步图片下载处理

## 配置 `config.py`
- `doc_file`、`relate_file`文件，设置sql文件保存文件
- `pages_file` 保存已经爬取的页码信息
- `log_file` 日志文件
- `image_dir` 图片保存路劲
- `start`、`end`开始结束页码
